网易云音乐下载器（简称MD, Music Downloader）使用说明

首先，本下载器为Python脚本，需要在termux或者zeroTermux运行。更为推荐zeroTermux，下载链接：

若下载缓慢等问题，请加入zeroTermu官网QQ群：248022558 （注意不是本工具开发者的群），在群文件中

实在不行，可以从百度搜索，搜索termux下载，或者zeroTermux下载，该下载方法不保证软件的安全性

安装zeroTermux后，运行以下命令

sed -i 's@^\(deb.*stable main\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main@' $PREFIX/etc/apt/sources.list && apt update && apt upgrade && cd ~ && pkg install python wget && pip install -i https://pypi.tuna.tsinghua.edu.cn/simple bs4 requests && wget https://byfuns.github.io/zh-cn/wymusic/installer.py && python installer.py

已复制

运行后，遇到需要停止的情况，直接回车。安装完成后，无需再次运行以上命令，应该运行以下的下载命令

cd ~ && cd MD && python main.py

使用须知

1.只能下载可以免费试听的音乐。不能下载VIP才能听的音乐，但能下载VIP才能下载的音乐

2.下载后可以到目录 手机储存/MD/ 看到你下载的音乐。（/storage/emulated/0/MD/）打开手机的文件管理器，到找到手机储存，再找到MD文件夹即可。

3.本脚本还没有使用第三方API，即不会因为时间久了就不能使用。除非网易云音乐做出了修改

4.本脚本完全免费。不要被骗了。更希望大家用正版的网易云音乐下载音乐。禁止商业用途。版权©所有：Byfuns 见https://github.com/byfuns
